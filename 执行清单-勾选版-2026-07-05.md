## 我的防 Bug 工作协议（无论你选哪些，都按这个执行）

1. **逐项独立补丁**，每项可单独回滚；改完一项立即在 375 / 768 / 1440 三档视口实际渲染截图对比。
2. **内容可见性永不依赖 JS**：关掉 JS 内容照常显示（A-M2 保险丝兜底），动效只是锦上添花层。
3. **只用 transform / opacity / clip-path**，零 CLS；`prefers-reduced-motion` 全局豁免并截图验证。
4. 改动只落在这些文件：4 个静态 HTML、`css/`、新增 `js/hm-motion.js`、`functions/_lib/cms-core.js` 的渲染函数与内联 CSS（选 C-M7 才碰 repository/[slug].js，选 E1 才碰 build 脚本）。**绝不碰**：鉴权、API、D1 迁移、净化器。
5. 交付前跑 `npm test`（52 项）+ 375px 横向溢出扫描 + 全页面回归截图。
6. **交付物**：修改后完整 zip + `CHANGELOG-给GPT.md`（改动文件清单、每项对应编号、部署注意事项、回滚方法）+ 前后对比截图包。
