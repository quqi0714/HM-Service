# Huamei 内容治理目录

本目录保存官网 AI-Ready 内容治理的工作底稿，不直接进入生产构建。

- `official-fact-sheet.md`：公司与服务事实候选值。只有状态为“已确认”的字段，才可以作为全站和外部平台的统一口径。
- `claims-register.md`：政策、金额、时效和绝对化主张的来源与处理状态。
- `cms-publishing-checklist.md`：每次发布或更新公寓、政策文章时使用的检查清单。

更新规则：

1. 不凭记忆覆盖事实；记录来源、负责人和审核日期。
2. 政策主张优先链接主管机构的一手页面。
3. 标记为“待业务确认”或“待修正”的内容，不自动同步到公开页面。
4. 外部平台（GBP、Yelp、GSC、Bing、GA4）完成核对后，在对应字段补充证据链接或截图路径。
