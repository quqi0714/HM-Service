# Huamei CMS 发布清单

## 内容与事实

- [ ] 标题直接回答用户正在寻找的问题。
- [ ] 首段讲清“是什么、适合谁、何时、哪里、怎么做”。
- [ ] 政策文章已填写适用范围、最后审核日、作者、审核人和官方来源。
- [ ] 截止日、租金、AMI、补助金额、房型和名额已按一手页面复核。
- [ ] 出现“保证、一定、永远、不会、免费、当天、最高”等词时已人工复核。
- [ ] 内容没有收集或要求业务不需要的医疗、健康或其他敏感信息。

## SEO / GEO

- [ ] 分类、关键词、slug、canonical、SEO 标题与描述已填写。
- [ ] 封面 alt 准确描述画面，不堆关键词。
- [ ] 封面图没有烧死会快速过期的日期或数字；如有，已安排下架/替换。
- [ ] 页面可见的来源、日期、作者与 BlogPosting/FAQPage 数据一致。
- [ ] 只有用户可见的 FAQ 才进入 FAQPage。

## 发布后

- [ ] 详情页返回 200，标题、正文、图片与来源模块可见。
- [ ] 列表页可以到达详情页，筛选参数页为 noindex,follow。
- [ ] sitemap 包含新 URL，lastmod 正确。
- [ ] IndexNow 任务已记录 URL、时间、响应或失败原因。
- [ ] 失败通知可重试，但没有阻塞 CMS 保存。
- [ ] 桌面与手机端没有水平滚动；键盘可完成主要操作。
